# Air Quality Index Checker

The Air Quality Index (AQI) Checker is a web application built with React and styled using Bootstrap that allows users to check the current air quality index of a specific city. By entering the name of the city, users can quickly obtain information about the air quality, helping them make informed decisions to protect their health and well-being.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

### `npm start`
